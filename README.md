# Form Submission

## Description

A contact form page that collects user information including name, surname, email, phone number, gender, and a message. The form submits data via POST to a PHP backend.

## Tech Stack

- HTML5
- CSS3 (inline styles)
- PHP (form action target)

## Features

- Form fields for name, surname, email, phone, gender, and message
- Required field validation using HTML5 attributes
- Radio button input for gender selection
- Textarea for longer message input
- Background image styling

## How to Run

1. Clone this repository
2. Serve `form.html` through a PHP-enabled web server (e.g., XAMPP, WAMP)
3. Ensure a `submit_form.php` script is available on the server to handle form submission
